## Installation

1. Clone the repo
```sh
git clone https://github.com/PPetkov2000/Mobile-Store
```
2. Install NPM packages
```sh
npm install
```

## General Requirements

- React.js for client-side
- Node.js
- Express framework 
- MongoDB Atlas for database
